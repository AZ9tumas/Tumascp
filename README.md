# Tumascp
**NOTES:**
**1. MakeFile works only on Linux.**
**2. This version of Tumascp required (GCC) 11.1.0**

# Making the Binary Executable
Use the following commands in order to start scripting in tumascp:
1. `make`
2. `./tumascp`

Running Files -> `./tumascp file_name`
